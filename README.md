# Impact of OPEC regulations on oil prices

Cílem projektu je zjistit, jak se cena ropy bude vyvíjet při různých regulačních opatřeních OPEC .  Modelujme jako opakovanou hru tří hráčů , kteří představují archetypy členských států. Každému hráči jsme určili kvótu, cenu těžby a úroveň trpělivosti a také strategii chování na trhu.
